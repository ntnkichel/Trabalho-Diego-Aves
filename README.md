# 🐦 Jogo das Aves (Lógica Proposicional)

Projeto desenvolvido para descobrir qual ave foi escolhida utilizando o menor número de perguntas possível.

---

## 🧠 Ideia do Projeto

O jogo consiste em:

* Um conjunto de *16 aves (A até P)* organizadas em uma matriz 4x4
* O usuário escolhe uma ave mentalmente
* O sistema faz *4 perguntas (sim/não)* para descobrir qual é

💡 Estratégia utilizada:

* Divisão em *linha (2 perguntas)*
* Divisão em *coluna (2 perguntas)*


---

## 🚀 Tecnologias utilizadas

* Python
* Flask
* HTML
* CSS

---

## 🖼️ Interface

O sistema exibe:

* A imagem com as aves
* Perguntas interativas
* Resultado final
* Botão de reinício

---

## ⚙️ Como rodar o projeto

1. Instale o Flask:


pip install flask


2. Execute o projeto:


python app.py


3. Acesse no navegador:


http://127.0.0.1:5000
